# Hotel Management System (Quiz Assignment)
This is a simple C# Console Application developed as a programming assignment for the Software Engineering department. It demonstrates the fundamental concepts of C# and basic management system logic.

## Features
Quiz Logic: A structured question-answer system developed for academic purposes.

Console Interface: Simple and effective command-line interaction.

C# Fundamentals: Use of variables, loops, and conditional logic.

## Technologies Used
Language: C#

Platform: .NET Console App

IDE: Visual Studio 2022

## How to Run
Clone this repository to your local machine.

Open the quiz.sln file in Visual Studio.

Press F5 or the Start button to run the application.
